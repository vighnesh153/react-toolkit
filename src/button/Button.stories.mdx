# Button

Primary UI component for user interaction. Read more about using the base Material UI
button [here](https://material-ui.com/components/buttons).

### Import
```tsx
import { Button, ButtonProps, ThemeProvider, theme } from "@vighnesh153/react-toolkit";
```

> You don't have to wrap all your buttons with `ThemeProvider`. You just need to do it once at the
> top-most level in your app or at different places where you want to use a different theme.

<Preview withToolbar>

    />
  </ThemeProvider>
